# ⚖️ Prompts Juridique

## 1. Analyse de Contrat

```
Analyse ce contrat et identifie les points d'attention.

**Type de contrat :** [Prestation / Licence / Travail / ...]
**Mon rôle :** [Client / Fournisseur]

**Clauses à analyser :**
[EXTRAITS OU CONTRAT COMPLET]

Analyse :
1. Clauses favorables
2. Clauses défavorables
3. Clauses manquantes
4. Risques identifiés
5. Points de négociation suggérés
```

---

## 2. Rédaction de Clause

```
Rédige une clause contractuelle pour : [BESOIN]

**Type :** [Confidentialité / Responsabilité / Résiliation / ...]
**Contexte :** [SITUATION]
**Protection souhaitée :** [CE QUE JE VEUX GARANTIR]

Propose :
1. Version standard
2. Version renforcée (en ma faveur)
3. Explications
```

---

## 3. CGV / CGU

```
Rédige les [CGV / CGU] pour : [ACTIVITÉ]

**Type d'activité :** [E-commerce / SaaS / Prestation / ...]
**Cible :** [B2B / B2C]
**Spécificités :** [PARTICULARITÉS]

Sections obligatoires :
1. Mentions légales
2. Objet
3. Prix et paiement
4. Livraison/Exécution
5. Rétractation (si B2C)
6. Garanties
7. Responsabilité
8. Données personnelles
9. Litiges
```

---

## 4. Conformité RGPD

```
Crée la documentation RGPD pour : [TRAITEMENT]

**Données collectées :** [TYPES]
**Finalités :** [OBJECTIFS]
**Durée conservation :** [DURÉE]

Documents à produire :
1. Registre de traitement (entrée)
2. Mention d'information
3. Formulaire de consentement (si nécessaire)
4. Clause contrat sous-traitant
```

---

## 5. Mise en Demeure

```
Rédige une mise en demeure pour : [SITUATION]

**Destinataire :** [NOM/SOCIÉTÉ]
**Fait reproché :** [DESCRIPTION]
**Action demandée :** [CE QU'ON VEUT]
**Délai :** [X jours]

Ton : Ferme mais professionnel.
Mentionner les conséquences en cas de non-exécution.
```

---

## 6. Politique de Confidentialité

```
Rédige une politique de confidentialité pour : [SITE/APP]

**Données collectées :**
- [Type 1]
- [Type 2]

**Finalités :** [USAGES]
**Cookies :** [Oui/Non, types]
**Transferts hors UE :** [Oui/Non]

Format : Clair, accessible, conforme RGPD.
```

---

## 7. NDA (Accord de Confidentialité)

```
Rédige un NDA pour : [CONTEXTE]

**Parties :** [QUI]
**Objet :** [INFORMATIONS À PROTÉGER]
**Durée :** [X ans]

**Unilatéral ou Bilatéral :** [TYPE]

Clauses clés :
1. Définition des informations confidentielles
2. Obligations
3. Exclusions
4. Durée
5. Sanctions
```

---

## 8. Résumé Juridique

```
Vulgarise ce texte juridique pour un non-juriste.

**Texte :**
[TEXTE JURIDIQUE]

**Public :** [Manager / Client / ...]

Format :
1. Ce que ça dit (en simple)
2. Ce que ça implique concrètement
3. Points de vigilance
4. Actions requises
```
